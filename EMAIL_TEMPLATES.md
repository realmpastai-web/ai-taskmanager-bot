## 🚀 Deploy in 3 Easy Steps

### Step 1: Get Discord Bot Token (2 min)
1. Go to https://discord.com/developers/applications
2. Create New Application → Add Bot
3. Copy the token (keep it secret!)

### Step 2: Deploy to Railway (2 min)
1. Go to https://railway.app
2. New Project → Deploy from GitHub
3. Add your Discord token as environment variable
4. Deploy - your bot is now online!

### Step 3: Invite to Server (1 min)
1. In Discord Developer Portal → OAuth2 → URL Generator
2. Select scopes: `bot`, `applications.commands`
3. Select permissions (see README)
4. Copy URL, open in browser, invite to your server

**Q: My bot shows "offline"**
